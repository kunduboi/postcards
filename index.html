3. Once you've successfully added cities, their airport codes should show up on the front of the card.

4. If you're happy with the list of cities, click the "Generate" button to generate a personalized postcard!

I'm currently leaving this button (and function) unlocked in order to get feedback on the algorithm and recommendations to improve it. Ideally, the button only let's you generate one postcard for one set of cities.
